# microistio
Istio 微服务

## 最佳实践

>Istio

https://istio.io/

- 服务框架：待补充
- 服务发现/负载均衡：Envoy Sidecar
- 边界代理/网关：Istio Gateway
- 调度和发布：K8s
- 配置中心：ConfigMaps
- 流量容错：启用Timeout, Retries & Outlier Detection，对应用是非侵入的
- 流量治理：A/B, Canary, Mirroring
- 安全治理：Citadel
- 监控治理：从Envoy采集，使用 kiali 可视化监控工具展示
- 链路追踪：Jaeger
- 指标监控：Prometheus
- 日志监控：EFK
- 数据库：MySQL
- NoSQL数据库：MongoDB
- 缓存：Redis
- 消息队列：Kafka
- 搜索引擎：ElasticSearch
- 文件存储：阿里云存储 OSS
